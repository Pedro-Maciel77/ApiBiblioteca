# Atividade Node + TypeORM (Livro)

Instruções rápidas:
1. Instale dependências: `npm install`
2. Rode em desenvolvimento: `npm run dev`
3. Endpoints:
   - POST /api/livros
   - GET /api/livros
   - GET /api/livros/:id
   - PUT /api/livros/:id
   - DELETE /api/livros/:id

Banco: SQLite (database.sqlite) gerado automaticamente.
