# 📚 API de Biblioteca

API desenvolvida em TypeScript para gerenciamento de uma biblioteca, permitindo o controle de livros, usuários e empréstimos de forma organizada e eficiente.

🚀 Objetivo

Esta API foi criada com o objetivo de praticar conceitos de:

Desenvolvimento de APIs REST

TypeScript no back-end

Organização de código com boas práticas

Manipulação de dados (CRUD)

🛠️ Tecnologias Utilizadas

Node.js

TypeScript

Express

ts-node

Nodemon

SQlite
